AES¶

LELEC2770 : Privacy Enhancing Technologies

AES key representation library

class aes.AES_key(rep_bytes)¶

AES Key with multiple representations

The supported formats are:

  • bytes: byte string (bytes type)
  • int: integer (int/long)
  • bin_str: string of bits ‘0’/‘1’
  • bin: list of bits [0/1]
  • hex: string of hex digits (lowercase)

The conversion between sequence and integer representations is in MSB order.

El Gamal¶

LELEC2770 : Privacy Enhancing Technologies

El Gamal encryption library

class elgamal.ElgamalCiphertext(p, c1, c2)¶

El Gamal ciphertext.

Thanks to group homomorphism, operations over plaintexts can be implemented in the ciphertext domain. For two ciphertexts ca, cb and an integer a, * ca+cb corresponds to plaintext addition (hence ciphertext multiplication) * a*ca corresponds to plaintext multiplication (hence ciphertext exponentiation)

Available operations: addition, subtraction, negation, multiplication.
